# Marché de Noël à Strasbourg

Strasbourg, la capitale de Noël, vous propose de nombreuses activités pour rendre votre visite au Marché de Noël inoubliable. Voici quelques idées pour profiter de cet événement qui attire chaque année des millions de visiteurs.

## Les Activités Incontournables

- **La Grande Roue** : pour une vue panoramique sur la ville.
- **Les Spectacles sur Scène** : les concerts, les spectacles de danse et les performances de théâtre.
- **Les Ateliers de Noël** : pour créer vos propres déco et objets de Noël.
- **Les Promenades en Calèche** : pour une balade romantique dans la ville.
- **Les Visites Guidées** : pour découvrir les secrets de la ville et de son histoire.

## Les Activités pour les Enfants

- **Le Village des Enfants** : des activités ludiques pour les petits.
- **Les Ateliers de Noël** : les enfants pourront créer leurs propres objets de Noël.
- **Le Père Noël** : pour une photo souvenir unique.
- **Les Spectacles de Noël** : des spectacles de marionnettes et des contes de Noël.

## Les Activités pour les Adultes

- **La Dégustation de Vins** : pour découvrir les vins d'Alsace.
- **La Dégustation de Bière** : pour découvrir les bières de Strasbourg.
- **Les Concerts de Jazz** : pour une soirée en musique.
- **Les Spectacles de Théâtre** : pour une soirée culturelle.

## Les Informations Pratiques

- **Dates** : le Marché de Noël de Strasbourg se déroule chaque année du 24 novembre au 31 décembre.
- **Horaires** : le Marché de Noël est ouvert tous les jours de 11h00 à 20h00.
- **Prix** : les billets sont gratuits mais certaines activités sont payantes.
- **Accès** : le Marché de Noël est accessible en tramway (ligne A ou D) ou en bus (lignes 2, 4, 6, 15, 30, 37, 71, 72).
